# 🐕 Patrulha Rural com Cães — Simulação Tática

**PMES / Canil** — Aplicativo PWA de simulação tática de patrulha rural com cães.

## Funcionalidades

- 6 cenários táticos: Embarcado, Coluna, Emboscada (Frente, Esquerda, Direita, Retaguarda)
- Drag & drop de unidades com mouse ou toque
- Movimento em grupo (COMANDO / APOIO)
- Transições animadas entre cenários
- Rotação individual de unidades
- Funciona offline (PWA / Service Worker)
- Otimizado para celular (landscape)

## Unidades

| Cor | Tipo | Membros |
|-----|------|---------|
| 🟢 Verde | Grupo COMANDO | CMT, S1, S2, Rast |
| 🟢 Verde | Grupo APOIO | AL1, SUB, PR2, PR1 |
| 🔵 Azul | Viaturas | RP3030, RP3031 |
| 🔴 Vermelho | Ameaça | V1, V2 |

## Instalação no celular

Acesse o link do GitHub Pages → menu do navegador → **"Adicionar à tela inicial"**

## Tecnologias

HTML5 + Canvas · CSS3 · JavaScript puro · PWA (Service Worker)
